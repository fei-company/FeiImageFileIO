# FeiImageFileIO

Python libraries for cryoEM and TIA related image formats. 
Python 2.7 + Numpy is prerequisite.

===================

Installation
python setup install

===================

Example 1
python FeiMrc2Img.py -i Paracetamol_dataset1_/ -o output_folder/image_ -x 1047 -y 937 -a 1 -F
